# ❄️ Frost Expense Tracker

A simple web app to track daily expenses. Built with HTML, CSS and javascript

## Features
- Add an expense (description, amount, category)
- View all expenses in a table
- Delete any expense
- Filter expenses by category
- See the live total amount spent
- Expenses are saved in the browser (using localStorage) — they stay even after closing/reopening the page
- Dark theme with grey-black-white colors + subtle icy/frost cyan accents

## How to Use
1. Open the file `index.html` in any web browser
2. Fill in:
   - Description (e.g. "Lunch")
   - Amount (e.g. 450)
   - Category (choose from dropdown)
3. Click **Add Expense**
4. To delete: click the red **Delete** button next to any item
5. To filter: use the dropdown at the top
6. Total updates automatically

No installation needed — works offline after first load.
